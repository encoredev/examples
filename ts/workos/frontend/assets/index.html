What is WorkOS?

Enterprise authentication with AuthKit hosted UI, SSO, and directory sync.

How auth works

Redirect to AuthKit, callback exchanges code for token, JWT verified by backend.

Encore integration

Auth handler verifies JWT via JWKS. getAuthData() provides user data to endpoints.
