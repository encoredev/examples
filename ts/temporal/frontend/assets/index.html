What is Temporal?

Temporal is a durable execution platform. Workflows are regular code with automatic retries, timeouts, and crash recovery.

This example

An order processing workflow:

  • Create order in database
  • Check inventory
  • Process payment
  • Ship order
  • Send confirmation

If shipping fails after payment, the workflow refunds automatically (saga pattern).

Encore integration

Activities call Encore services via type-safe clients. The orders service uses SQLDatabase for persistence.

Try it

Use out-of-stock as Product ID for inventory failure. Use fail-shipping for the saga refund path.
