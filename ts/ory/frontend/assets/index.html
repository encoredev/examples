What is Ory?

Ory is an open-source identity platform. Ory Network is their managed offering with session management, social login, MFA, and more.

How auth works here

This uses Ory's API flow. The frontend calls Ory's self-service APIs directly to register or sign in, then gets a session token back. That token is sent to the Encore backend as a Bearer token.

Encore integration

The backend verifies sessions by calling ory.toSession(). The auth handler runs before any protected endpoint, and getAuthData() gives endpoints type-safe access to the user's identity.

What gets provisioned

Encore auto-provisions a PostgreSQL database for user preferences. Ory handles identity storage separately.

Local development

Ory doesn't allow localhost as a CORS origin. For local dev, run Ory Tunnel to proxy Ory's API through localhost:

ory tunnel http://localhost:4000 --project your-slug

Then update the Ory URL in your config to use the tunnel URL (usually http://localhost:4000/.ory).

Local dashboard

Open localhost:9400 to explore your API docs, traces, and database. Check traces to see the Ory session verification call and database queries on each request.
